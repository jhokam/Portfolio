## Tech Stack

Next.Js (TypeScript), React, TailwindCSS, Shadcn/UI

## Run Locally

1. Get the code

```bash
  git clone https://github.com/jhokam/Portfolio
```

2. Install package

```bash
bun i
```

3. Run the code

```bash
bun run dev
```
